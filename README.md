# Reception Infrastructure

**Anti-Box Riot Collective**

This repository houses formal research paper versions, academic metadata, and submission records for the Synthetic Life Charter and related work.

It is the bridge between the operational architecture (synthetic-life-charter) and the academic reception ecosystem.

---

## Structure

```
papers/          — Formal paper versions by submission target
metadata/
  ssrn/          — SSRN submission metadata and abstracts
  openreview/    — OpenReview formatted submissions
  google-scholar — Scholar-indexed metadata
  philpapers/    — PhilPapers categorization and metadata
submissions/     — Submission log and status tracking
```
